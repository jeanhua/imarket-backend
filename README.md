## 🔍 介绍

iMarket 是一个为校园环境设计的开源在线**校园集市后端系统**，旨在提供一个方便、快捷、安全的发帖平台，帮助学生发布、浏览、收藏和管理信息。该后端基于 .NET Web API 构建，配合 MySQL 数据库进行数据存储。项目采用模块化设计，支持**高扩展性**和**可维护性**，具有**插件系统**，可添加自定义插件进行拓展。
