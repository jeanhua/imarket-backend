## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/jeanhua/imarket-backend.git

# 安装依赖
dotnet restore

# 配置数据库（修改appsettings.json）
{
    "ConnectionStrings": {
        "DefaultConnection": "Server=localhost;Database=imarket;User Id=imarket_root;Password=your_password;"
    }
}

# 运行项目
dotnet run
```
